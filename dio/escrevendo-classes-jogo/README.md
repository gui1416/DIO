# ⚔️ Escrevendo as Classes de um Jogo

Projeto desenvolvido como parte do desafio **Escrevendo as Classes de um Jogo** da **Digital Innovation One (DIO)**.

O objetivo é praticar os conceitos fundamentais da Programação Orientada a Objetos (POO) em JavaScript, utilizando classes, objetos, métodos e estruturas condicionais para representar heróis de uma aventura e suas diferentes formas de ataque.

---

## 📖 Sobre o desafio

Foi criada uma classe genérica chamada **Hero**, responsável por representar um personagem de um jogo.

Cada herói possui as seguintes propriedades:

* Nome
* Idade
* Tipo

Os tipos disponíveis são:

* Mago
* Guerreiro
* Monge
* Ninja

A classe também possui o método `atacar()`, que exibe uma mensagem diferente de acordo com o tipo do herói.

### Tipos de ataque

| Tipo      | Ataque         |
| --------- | -------------- |
| Mago      | Magia          |
| Guerreiro | Espada         |
| Monge     | Artes marciais |
| Ninja     | Shuriken       |

Ao executar o método `atacar()`, é exibida a mensagem:

```text id="hpt8jq"
O {tipo} atacou usando {ataque}
```
